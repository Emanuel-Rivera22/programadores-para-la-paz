Pregunta 1
¿Qué es un sistema operativo?

A. Un programa que navega en internet

B. El software que administra el hardware y permite ejecutar programas

C. Un archivo de texto

D. Un servidor web

RTA/: B. El software que administra el hardware y permite ejecutar programas.

Pregunta 2
¿Cuál de los siguientes es un sistema operativo de código abierto?

A. Ubuntu

B. Windows

C. iOS

D. Chrome

RTA/: A. Ubuntu.

Pregunta 3
¿Qué es la terminal en un sistema Linux?

A. Una aplicación para editar imágenes

B. Una interfaz que permite ejecutar comandos directamente en el sistema

C. Un navegador web

D. Un antivirus

RTA/: B.  Una interfaz que permite ejecutrar comandos directamente en el sistema.

Pregunta 4
¿Cuál de las siguientes prácticas mejora la seguridad digital?

A. Compartir contraseñas con todos

B. Usar contraseñas fuertes y proteger credenciales

C. Guardar claves en archivos públicos

D. Desactivar controles de seguridad

RTA/: B. Usar contraseñas fuertes y proteger credenciales.
